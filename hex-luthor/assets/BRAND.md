# HEX Luthor Brand Guidelines

## Logo

<img src="logo-original.png" width="200" alt="HEX Luthor Logo">

The HEX Luthor logo is a minimalist line-art face with RGB eyes, symbolizing:
- **The face** → Human creativity, personal expression
- **RGB eyes** → Color, hex codes, digital precision
- **Minimalist style** → Clean, borderless, seamless

## Colors

| Token | Hex | Usage |
|-------|-----|-------|
| Primary Dark | `#0d1117` | Default background |
| Secondary Dark | `#161b22` | Card backgrounds |
| Accent | `#58a6ff` | Links, highlights |
| Text Primary | `#f0f6fc` | Headings |
| Text Secondary | `#8b949e` | Body text |

## Usage

- Use the logo at minimum 48px width
- Maintain aspect ratio
- Place on dark backgrounds for best visibility
- The RGB eyes should never be recolored

## Favicon

Use `logo-original.png` cropped to a square for favicon generation.
