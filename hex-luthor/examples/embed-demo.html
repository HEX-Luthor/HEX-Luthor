🚀 What is Simple Embed?

The Simple Embed is a drop-in script that lets anyone add a HEX Luthor background to their page with zero configuration. Just add one <script> tag with data-* attributes.

📋 Usage

Copy and paste this into your HTML:

<script src="https://cdn.jsdelivr.net/npm/hex-luthor@latest/dist/embed.min.js" 
  data-pattern="gradient" 
  data-colors="#0d1117,#161b22"
  data-target="body"
  data-text="My Project"
  data-subtext="A cool description"
  data-animation="true">
</script>

⚙️ Available Attributes

data-pattern dots, grid, diagonal, hexagon, noise, waves, mesh, gradient, aurora
data-colors Comma-separated hex colors (e.g., "#0d1117,#21262d")
data-target CSS selector for target element (default: "body")
data-text Optional banner text overlay
data-subtext Optional subtitle text
data-animation Set to "true" for animated background
data-opacity Pattern opacity 0-1 (default: 0.8)
data-size Pattern tile size in pixels (default: 64)

🎨 Pattern Examples

Try these patterns by changing the data-pattern attribute:

  • dots — Subtle dot matrix
  • grid — Fine grid lines
  • diagonal — Diagonal stripes
  • hexagon — Honeycomb pattern
  • noise — Fractal noise texture
  • waves — Flowing wave lines
  • mesh — Connected node mesh
  • gradient — Smooth color blend
  • aurora — Northern lights effect
